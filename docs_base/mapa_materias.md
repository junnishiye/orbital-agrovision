# Mapa de Integração das Matérias

| Matéria | Como entra no projeto |
|---|---|
| Pensamento Computacional | Sistema Python com matriz de dados, ciclos, alertas e relatório |
| Data Structures | Menu interativo para inserir, visualizar e analisar dados |
| Prompt and AI | IA interpreta os dados e recomenda ações |
| Energias Renováveis | Análise de bateria, geração solar, economia de energia e sustentabilidade |
| Computer Organization | Simulação IoT com sensores de temperatura, luminosidade e vibração |
| Computer Science | Lógica booleana para acionar alerta geral |
| Modelagem Matemática | Modelo de risco por temperatura e estresse operacional |
| Modelagem Linear | Análise estatística com base real |
